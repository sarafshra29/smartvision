# smartvision
SmartVision AI is an advanced Computer Vision and Deep Learning project designed to perform intelligent multi-class object recognition using state-of-the-art CNN architectures and YOLOv8 object detection. The system combines transfer learning, real-time object detection, and a user-friendly Streamlit web application to deliver accurate and scalable AI-powered visual recognition solutions.

🚀 Features
Multi-class image classification using:
VGG16
ResNet50
MobileNetV2
EfficientNetB0
Real-time object detection using YOLOv8
Detection of 25 diverse COCO dataset classes
Bounding box visualization with confidence scores
Model comparison dashboard with performance metrics
Streamlit-based interactive web application
Cloud deployment support via Hugging Face Spaces
Optimized inference for real-world applications
